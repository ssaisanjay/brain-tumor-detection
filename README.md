Developed an automated brain tumor classification system using VGG16-based transfer learning on MRI images to classify glioma, meningioma, pituitary tumor, and no tumor with high accuracy. Applied data preprocessing, normalization, and augmentation techniques to improve generalization and robustness. Achieved F1-scores above 0.85 across all classes and implemented a real-time prediction function with confidence scores. Explored future enhancements such as model interpretability using Grad-CAM, multimodal data integration, and deployment for real-time clinical diagnostics.
